# 基础部分 <br/>

## 状态: <br/>
### 命令模式(开机默认):<br/>
> 蓝色LED以10Hz频率闪烁,可以通过命令传输测量数据,访问传感器的功能 <BR/>

### 流模式 <br/>
> LED蓝/红淡入淡出,用于数据采集,无法设置传感器参数 <br/>

## 属性: <br/>
> 右手笛卡尔坐标系<br/>
>
> 地磁北极方向为X轴正方向;<br/>
>
> 磁西为Y轴正方向;<br/>
>
> 向上为Z轴正方向(与重力方向相反);<br/>
> 旋转：yaw(偏航)：绕Z轴360度旋转;<br/>pitch(俯仰)：绕Y轴左右各90度;<br/>roll(桶滚)：绕X轴360度旋转;<br/>
